Convert Unicode to Character in Python

We can easily convert Unicode to character using the chr method in Python.

Conversion

Using chr method in Python, we can convert Unicode to a character as below.

chr(97)
# a

chr(98)
# b

By the way, use ord method to convert character to a Unicode.

ord('a')
# 97
