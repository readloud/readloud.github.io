ROT13, ROT47

ROT13 (rotate by 13 places), ROT47 (rotate by 47 places) are simple letter substitution ciphers.

Decode

CyberChef is useful online tools to decode ROT13, ROT47.


Encode

ROT13

echo hello | tr 'A-Za-z' 'N-ZA-Mn-za-m'
